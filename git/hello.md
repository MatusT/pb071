## Hello

```
mkdir hello                   # vytvorenie adresára
cd hello                      # vstúpenie do nového adresára
nano main.c                   # vytvorenie zdrojového kódu, (možné použiť aj klasický editor)
git add main.c                # pridanie súboru "main.c" do fázy STAGE
git commit -m "Úloha Hello"   # 
git push                      #
```
